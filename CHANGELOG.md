# 変更履歴

## 1.1.0

- イベントリレーへの自動追従機能を追加
  - 放送波のイベントリレー情報を検出し、番組終了時刻にリレー先チャンネルへ自動追従
  - イベントリレー情報を送出しない放送局向けに、番組名の完全一致と放送時刻の連続からリレー先を推定する機能を追加（デフォルトでは無効）
  - 自動追従時にタイルの位置、フォーカス、音量、ミュート、ラージ / スモール別の再生設定を維持
- `[` / `]` キーで音量を変更した際、マウス位置にかかわらず操作パネルとチャンネル情報を表示するように改善

## 1.0.1

- チャンネル選択画面の LIVE ラベル判定で、番組名の途中に `[生]` や `生中継` が含まれる場合も生放送として扱うように修正
- フルスクリーン中に選局画面を Esc で閉じようとするとクラッシュすることがある問題を修正
- フルスクリーン中にタイル配置を変更して終了した次回起動時、ウインドウサイズが新しいタイル配置の比率に合わないことがある問題を修正
- タイル配置変更時、可能な限りタイルの表示倍率を維持してウインドウサイズを調整するように修正
